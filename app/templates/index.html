EasyPosting API

This is an API, built with Python, Flask and Love, that can help you to easily add announcements to your website.

This can be done in a few simple steps:

Create an account

Just go to the Sign Up Page and register.

Sign Up

Post your first announcement

Then our tool below to.


Verify it worked

Open your browser in "https://easyposting.heroku.app/read?username=<Your username>" or type your username on the input below and verify that your latest post is online.


Add our plugin to your site

Go to the GitHub repository of this project and read the README.md file to get instuctions of how to add our plugin to your website.
